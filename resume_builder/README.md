# 简历生成器

一个纯前端简历网页工具，用来填写信息、实时预览并导出 PDF。

## 主要功能

- 基本信息、教育经历、项目经历、技能清单编辑
- 右侧实时预览
- 照片上传与裁剪
- A4 导出
- 单页/双页切换（内容过多时可用）
- 本地自动保存（刷新不丢）

## 使用方式

### 直接使用

打开 `index.html` 即可。

### 本地启动（推荐）

```bash
python3 -m http.server 8000
```

浏览器访问：`http://localhost:8000`

## 导出 PDF

1. 点击页面里的“导出 PDF”
2. 打印目标选择“另存为 PDF”
3. 纸张大小选择 `A4`
