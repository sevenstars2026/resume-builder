# Resume Builder v2.5 Glass

一个可直接离线使用的前端简历生成器，支持实时预览、A4 导出、单页/双页模式、照片裁剪上传和本地自动保存。

## 功能

- 分模块编辑：基本信息、教育、项目、技能
- 实时预览：左侧编辑，右侧同步
- 导出 PDF：浏览器打印导出
- A4 适配：单页压缩与双页切换
- 照片裁剪：上传后拖拽+缩放裁剪
- 本地存储：刷新后内容不丢

## 本地使用

### 方式 1：直接打开

双击 `index.html` 即可使用。

### 方式 2：本地服务器（推荐）

```bash
python3 -m http.server 8000
```

然后访问 `http://localhost:8000`

## 给他人使用

### 1) GitHub 代码方式

```bash
git clone <你的仓库地址>
cd day6-resume-builder
python3 -m http.server 8000
```

### 2) 在线地址方式（推荐）

把项目部署到 GitHub Pages 后，别人直接打开链接即可使用。

## 导出建议

- 在浏览器打印中选择 `另存为 PDF`
- 纸张大小选择 `A4`
- 边距选 `默认` 或 `最小`
- 如内容较多可切到 `双页` 模式保证可读性
